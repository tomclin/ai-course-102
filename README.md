# 放手交給 AI——讓你的 AI 同事真正上工

**「AI 好同事三部曲」第二堂。** 120 分鐘實作工作坊：學會把一件事完整交給 AI 做完——授權、檢查點、責任邊界。

**📖 線上直接看：<https://tomclin.github.io/ai-course-102/>**

## 一句話說完這門課

> **放手，但不放養。**（Delegate, don't abdicate.）

上一堂（101）AI 認識了你；這一堂，把工作真正交出去——先看計畫、設好檢查點、對著標準驗收。交出去的是工作，沒交出去的是責任。

## 使用說明

- **課堂學員**：跟著講師走，每人先做 Level 1，再依程度選 Level 2 或 3。
- **自學者**：建議先完成第一堂（[讓 AI 認識你](https://tomclin.github.io/ai-course-101/)）的個人脈絡卡，再做本堂 Level 1。
- **列印**：題卡與工具表已針對 A4 排版，瀏覽器直接列印，每份一頁。
- **深／淺色**：頁面自動配合裝置模式；列印一律輸出淺色。

## 三個誠實的原則

1. **工具不限**——ChatGPT、Gemini、Claude、Copilot 都行，**所有練習皆可在免費版完成**（課堂含講師以付費工具的示範，示範不在此承諾內；免費功能對照查證日期：2026-07-12）。
2. **AI 會錯**——放手的前提是驗收；檢查點不是不信任 AI，是專業。
3. **責任不外包**——你交辦的是工作，出錯時面對世界的仍然是人。
